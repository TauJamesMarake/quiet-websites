# Leave Something Here

A quiet exchange.

## Concept
This website invites the visitor to leave something small behind —
a thought, a feeling, a few words.

In return, they receive something left by someone else.

No names.
No profiles.
No permanence.

Just passing something along.

## Features
- Short anonymous text input
- Gentle message exchange

- Local-only storage (localStorage)
- Light / Dark mode toggle
- Soft transitions and grain texture
- Fully responsive
- No analytics, cookies, or servers

## Privacy
Everything stays on the visitor’s device.

There is:
- No database
- No tracking
- No network requests

Each visit is private and temporary.

## Tech Stack
- HTML
- CSS
- Vanilla JavaScript

No frameworks.
No dependencies.
No build step.

## Why This Exists
Most interactions online are transactional.

This one is simply human.
