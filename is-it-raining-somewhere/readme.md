# Is It Raining Somewhere?

Yes.

## Concept
This website answers a question that doesn’t need checking.

At any given moment,
somewhere on Earth,
rain is falling.

The project exists to acknowledge scale —
that weather, like life, continues beyond our immediate view.

## Features
- Single, honest answer
- Slowly changing atmospheric messages
- Light / Dark mode toggle
- Subtle film-grain texture

- Fully responsive layout
- No APIs, tracking, or analytics

## Accuracy
No live weather data is used.

The answer is always correct.

## Tech Stack
- HTML
- CSS
- Vanilla JavaScript

No frameworks.
No dependencies.
No forecasts.

## Why This Exists
Not everything needs precision.
Some truths are true often enough to be trusted.
